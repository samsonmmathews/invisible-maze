# Invisible Maze

Open `index.html` in a browser to play the invisible maze game.

How to play

- Use the arrow keys to move the player (green dot).
- The maze walls are invisible by default. If you get stuck, toggle "Show maze" to reveal the layout for testing.
- Reach the far bottom-right cell to win.

Notes

- This is a static HTML page. No server is required - just open the file in a modern browser.
- The maze is randomly generated on each reset/page load.
